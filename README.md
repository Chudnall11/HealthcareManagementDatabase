# Healthcare Management Database System

## Description
This project is a healthcare management database system designed using entity relationship modeling (ERD). The database structure models relationships between patients, scheduling, referrals, doctors, contracts, insurance, employees, medical supplies, suppliers, and cost reporting.

The project demonstrates database design concepts used in healthcare management systems and focuses on relational schema organization, entity relationships, and data management workflows.

## Features
- Patient scheduling and appointment tracking
- Doctor referrals and service management
- Insurance and contract relationships
- Employee and management structure
- Medical supply and supplier tracking
- Cost reporting and management organization
- Relational database entity modeling

## Concepts Demonstrated
- Entity Relationship Diagrams (ERD)
- Relational Database Design
- Primary and Foreign Keys
- One-to-Many Relationships
- Database Normalization
- Healthcare Workflow Modeling
- Data Organization and Management
